# Python-Kod-denemeleri
Advanced Grade Calculation System (Tkinter GUI)
Temel aritmetik operatörleri, gelişmiş kullanıcı arayüzü (GUI) prensipleriyle birleştiren kapsamlı bir Python çalışması.
​Öne Çıkan Özellikler: Gerçek zamanlı veri doğrulama (Validation), Canvas ile dinamik görsel geri bildirim, hata yakalama (Exception Handling) ve modern UX tasarımı.
​Teknik Detay: Tkinter kütüphanesi üzerinde özel "Frame" mimarisi ve event-binding (olay bağlama) kullanılmıştır.

CyberPass Pro - Gelişmiş Şifre Üretici ve Güvenlik Analiz Aracı
CyberPass Pro, kriptografik olarak güvenli şifreler oluşturmak ve bu şifrelerin güvenlik seviyelerini analiz etmek için tasarlanmış profesyonel bir masaüstü uygulamasıdır. Tamamen Python standart kütüphaneleri kullanılarak geliştirilmiş olup, hiçbir ek kurulum veya bağımlılık gerektirmez.

Web Performance & Security QA Tool
Bu araç, bir web sitesinin bağlantı hızını, sunucu yanıt kodunu ve SSL güvenlik durumunu analiz eden bir Python uygulamasıdır.
​QA Yaklaşımı: Yazılım test süreçlerinde kullanılan "Site Sağlık Kontrolü" mantığıyla geliştirilmiştir.
​Teknik: requests kütüphanesi ile HTTP protokolü analizi ve Tkinter ile veri görselleştirme yapılmıştır.
​Amacı: Web sitelerinin performans ve güvenlik standartlarını hızlıca denetlemek.
