# Python-Kod-denemeleri
Advanced Grade Calculation System (Tkinter GUI)
Temel aritmetik operatörleri, gelişmiş kullanıcı arayüzü (GUI) prensipleriyle birleştiren kapsamlı bir Python çalışması.
​Öne Çıkan Özellikler: Gerçek zamanlı veri doğrulama (Validation), Canvas ile dinamik görsel geri bildirim, hata yakalama (Exception Handling) ve modern UX tasarımı.
​Teknik Detay: Tkinter kütüphanesi üzerinde özel "Frame" mimarisi ve event-binding (olay bağlama) kullanılmıştır.

CyberPass Pro - Gelişmiş Şifre Üretici ve Güvenlik Analiz Aracı
